Qworum for web pages

Qworum JavaScript library for use inside web pages on web browsers. Qworum is the Service Web that enables distributed web applications. See the Qworum website.

With this library web pages can:

  1. verify that the Qworum extension is installed and enabled on the end-user's browser.
  2. use Qworum's capabilities by communicating with the Qworum browser extension.

Usage

  1. First, copy and paste qworum-for-web-pages.mjs into your web frontend project directory.
  2. Then create a JavaScript file for your web frontend project and link it to your web page like this: <script type="module" src="./path/to/your.js"></script>. Example.
  3. After that, import the Qworum library like this in your.js: import { Qworum } from "./path/to/qworum-for-web-pages.mjs";. Example.
